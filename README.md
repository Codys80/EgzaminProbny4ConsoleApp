# Część I. Aplikacja konsolowa  
- Program ma dynamicznie poprosić użytkownika o podanie: 
  * imienia pierwszego ucznia oraz jego ocen 
  * imienia drugiego ucznia oraz jego ocen 
- Oceny powinny być podawane jako liczby całkowite od 1 do 6. Jeśli zostaną podane inne wartości mają zostać pominięte i ma się wyświetlić komunikat o tym, że są przyjmowane tylko liczby całkowite od 1 do 6. 
- Wpisanie wartości 0 ma zakończyć wprowadzanie ocen dla danego ucznia. 
- Dla każdego ucznia należy obliczyć sumę jego ocen. 
- Program ma obliczyć NWD obu sum przy wykorzystaniu algorytmu Euklidesa. 
- Program ma wyświetlić: 
  * imiona uczniów 
  * listy ich ocen 
  * sumy ocen
  * wartość NWD ich sum
  * oraz wskazać zwycięzcę (ucznia z większą sumą ocen)
- Po prezentacji wyniku program ma umożliwić użytkownikowi wybór:
   * ponownego porównania (nowi uczniowie i nowe oceny) 
   * lub zakończenia działania programu 
## Założenia: 
- Program najpierw pobiera dane o dwóch uczniach i ich ocenach. 
- Po zakończeniu wpisywania ocen, wyniki są sortowane i wyświetlane wraz z ich sumami. 
- Program oblicza NWD sum ocen i prezentuje go w wynikach, a następnie wskazuje zwycięzcę lub remis.   
- Po zakończeniu rundy użytkownik decyduje, czy chce kontynuować czy zakończyć działanie programu. 
- Zabronione jest korzystanie z gotowych funkcji
